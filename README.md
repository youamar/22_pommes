# 22 pommes
22 pommes est un jeu dont le but est d'être le premier à réunir exactement 11 pommes vertes et 11 pommes rouges dans ses paniers, ou d'obliger son adversaire à dépasser ce nombre.

|-- info.txt

    `-- tests.pro
